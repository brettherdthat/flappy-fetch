# Flappy Fetch

A fun Flappy Bird-style game featuring a golden retriever fetching tennis balls!

**Play the game here:** https://brettherdthat.github.io/flappy-fetch/

## About

Navigate your golden retriever through green pipe obstacles while collecting tennis balls for bonus points. The game starts easy and gradually increases in difficulty as your score goes up.

## How to Play

- **Click** or press **SPACE** to make the dog jump/flap upward
- Navigate through the gaps in the pipes
- Collect tennis balls for +5 bonus points
- Each pipe you pass gives +1 point
- Don't hit the pipes, ground, or ceiling!

## Features

- Progressive difficulty system
- Adorable golden retriever sprite
- Tennis ball collectibles
- Smooth Flappy Bird-style physics

## Running Locally

1. Clone this repository
2. Start a local web server in the project directory:
   ```bash
   python3 -m http.server 8000
   ```
3. Open your browser to `http://localhost:8000`
